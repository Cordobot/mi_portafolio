# Adrián Alvarez | Android Developer Portfolio 🚀

Este es mi portafolio profesional, diseñado para ofrecer una experiencia premium y moderna, destacando mis habilidades y proyectos en el ecosistema Android.

## 🌟 Características

- **Diseño Premium**: Estética moderna con glassmorphism, modo oscuro nativo y animaciones fluidas.
- **Multilingüe**: Soporte completo para Español e Inglés con cambio dinámico.
- **Arquitectura "No-Build"**: Implementado con HTML5, Tailwind CSS (CDN) y Vanilla JavaScript. Sin dependencias pesadas ni tiempos de compilación.
- **Responsive**: Totalmente optimizado para dispositivos móviles, tablets y escritorio.
- **Interactividad**: Efecto de máquina de escribir, modales de contacto y sistema de navegación suave.

## 🛠️ Stack Tecnológico

- **Core**: HTML5, CSS3, JavaScript (ES6+).
- **Styling**: Tailwind CSS, Custom Keyframe Animations.
- **Icons**: Lucide Icons.
- **Assets**: Imágenes optimizadas y SVGs.

## 📂 Estructura del Proyecto

- `index.html`: Estructura semántica y contenedores principales.
- `style.css`: Definición de animaciones, variables de diseño y clases personalizadas.
- `script.js`: Lógica de internacionalización, gestión de temas (dark/light) e interactividad.
- `assets/`: Recursos visuales y capturas de proyectos.

## 🚀 Despliegue

Al ser un sitio estático puro, puede ser desplegado instantáneamente en:
- GitHub Pages
- Vercel
- Netlify
- O simplemente abriendo `index.html` en cualquier navegador.
